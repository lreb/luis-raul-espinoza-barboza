Folder Arrange

Unzip the archive and find out the folder "buyer-file". There are all the template files in this folder. You can open the following files for viewing in browser.

  1. dist — folder with final build.
  2. node_modules — project dependencies.
  3. src/scss — folder with scss files.
  4. src/assets/images — folder with images and other files.
  5. src/js — folder with javascript files.
  6. src/index.html Index file
  7. .gitignore config for ignore git
  8. .prettierrc config for prettier
  9. webpack.config.js Webpack configuration file.
  10. package.json - Defines the project metadata, dependencies, and scripts.
  11. package-lock.json - Ensures consistent dependency versions across installations.

Installation

  • Open CMD / Terminal in Portify folder and run npm install for installing necessary dependencies
  • Make sure you have installed nodejs on your system

Start Dev Server

  • Open CMD / Terminal in Portify folder and run npm run dev for running this project on your machine locally.

Build Project

  • Open CMD / Terminal in Portify folder and run npm run build for make a production build for this project. it will create a folder named /build and final output will be there.

FTP Upload

  • Open Your FTP manager and connect to your hosting
  • Browse to required directory(Normally index.html)
  • Upload the files inside dist folder

Setup Contact Email

In this website we used email.js to send email direct to your gmail acount. Now you have to setup emailjs with your own gmail account.

To setup emailjs for this site, goto https://www.emailjs.com/

Then follow this steps :
  1. Goto Dashboard and collect your public key
  2. Then create a email service and collect service id
  3. Now create a email template and collect template id
